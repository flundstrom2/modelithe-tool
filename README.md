# Modelithe issue tracking tool

The Low-overhead [Modelithe Issue tracking tool and project management tool](https://modelithe.com) for growing companies, founders and personal use.

The Modelithe issue tracking tool is more than just a tool to track tasks, epics and features.

The Modelithe tool works just as well as a bug-tracking tool, or just as a personal todo-planner. But coupled with the Modelithe method, the Modelithe tool brings tracking into a complete project management system with transparency across the entire board.

All in order to fulfil the vision of Modelithe.
